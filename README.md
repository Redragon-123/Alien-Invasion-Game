# Alien-Invasion-Game
Alien Invasion Game based on Pygame

## 📖 Introduction
A 2D shooting game developed with Python Pygame, players need to shoot down aliens as much as they can to get a higher score.
Players have 3 lives and will lose a life if they were hit by aliens or aliens reach the bottom of screen.

The predecessor of this game is Alien_Invasion which is thaught in *Python Crash Course* by Eric Matthes. 

## 🎮 Game controls
Arrow keys or "W,A,S,D": Move spaceship.
Space: Fire
Lshift: Dash (with CD)
R: Reload (5 bullets reload one by one, you can use "Space" to interrupt.)
Esc: Pause the game and show menu.
Q: Directly quit the game.

## 👽 Aliens
Invader: Blue ship with no speciality.
Breaker: Red ship with lower speed which could fire bullets(could be counteracted by player's bullets).
Ghost: Green ship with higher speed which could be invisible for seconds.
